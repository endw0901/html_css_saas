# short cut command

```
// alt + shift + 下矢印・・・・行copy

.box*3 + enter => 
      <div class="box"></div>
      <div class="box"></div>
      <div class="box"></div>

- ctrl + 矢印：単語移動

- div#about + Enter

- クラス： .aaaaa + enter
- id: #bbbb + enter


// 応用
- section#home-info + enter => <section id="home-info"></section>
- a.btn.btn-light + enter => <a href="" class="btn btn-light"></a>

```
